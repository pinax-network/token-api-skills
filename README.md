# token-api-skills

Task-focused skills for AI agents using the Token API.

## Structure

- `skills/`: user-facing agent skills
- `shared/references/`: reusable API references and conventions

## Initial skills

- `wallet-portfolio-agent`
- `token-holder-agent`
- `transfer-investigation-agent`
- `dex-liquidity-agent`
- `network-status-agent`
- `api-integration-helper`

Each skill keeps workflow instructions in `SKILL.md` and UI metadata in
`agents/openai.yaml`.
