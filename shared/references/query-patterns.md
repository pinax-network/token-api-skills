# Query Patterns

## Wallet Portfolio

1. Identify chain family and network.
2. Fetch token balances.
3. Fetch native balance separately if needed.
4. Summarize holdings and notable concentrations.

## Holder Analysis

1. Resolve token contract or mint.
2. Fetch holders.
3. Highlight concentration and rank distribution.

## Transfer Investigation

1. Anchor on transaction hash, signature, address, or contract.
2. Add time or block filters to narrow scope.
3. Separate token transfers from native transfers when needed.

## Liquidity Analysis

1. Identify protocol and pool.
2. Fetch pool metadata.
3. Fetch swaps or OHLCV for activity context.

## Monitoring

1. Check health.
2. Check version.
3. Confirm supported networks.
